## 2. Executive Summary

This Statement of Work defines the scope, deliverables, responsibilities, assumptions, and acceptance criteria for the Azure Modernization Rapid Assessment engagement.

The objective is to analyze the customer workload inventory and provide a structured modernization recommendation draft with architecture rationale, prerequisites, and risk notes.

## 3. Objectives

- Establish a normalized inventory baseline for in-scope workloads
- Produce workload-level modernization recommendations using the 7R model
- Provide an executive summary for prioritization and next-step planning
- Create a validated draft that can feed migration planning

## 4. Scope of Services

### In Scope

- Data intake and inventory readiness validation
- Normalization of inventory into canonical workload schema
- Recommendation draft generation per workload
- Architect-led quality review and exception triage
- Final assessment readout session

### Out of Scope

- Migration execution and cutover activities
- Application code refactoring implementation
- Detailed target-state architecture implementation plans
- Production operations handover
- Any activities not explicitly listed in this SOW

## 5. Deliverables

- Deliverable 1: Normalized workload inventory dataset
- Deliverable 2: Workload-level modernization recommendation draft
- Deliverable 3: Executive summary and readout deck/notes
- Deliverable 4: Action-oriented next-step backlog
