# Azure Modernization Rapid Assessment

## Service Summary

Azure Modernization Rapid Assessment is a fixed-scope advisory engagement that converts your current application and infrastructure inventory into a prioritized modernization recommendation draft.

The service uses a structured assessment engine and Microsoft-aligned modernization practices to classify workloads, recommend target Azure paths, and identify prerequisites and risks.

## Business Outcomes

- Faster decision-making on modernization options
- Clear workload-by-workload recommendation baseline
- Reduced manual analysis effort for architecture teams
- Consistent deliverable quality across portfolios

## What We Analyze

- Application and server inventory exports
- Environment context (production, DR, dev, test)
- Platform signals (OS, database engine, footprint)
- Criticality and exposure indicators (public-facing, business impact)

## What You Receive

- Workload normalization output (standardized inventory view)
- 7R recommendation draft per workload
- Target Azure service suggestion per workload
- Rationale, prerequisites, and risk notes
- Executive summary for stakeholder alignment

## Engagement Model

### Typical Duration

1 to 2 weeks for initial assessment cycle, depending on data quality and portfolio size.

### Delivery Approach

1. Intake and data validation
2. Inventory normalization and recommendation generation
3. Architect review and quality gating
4. Executive readout and next-step planning

## Customer Responsibilities

- Provide source inventory export in agreed format
- Confirm scope boundaries and portfolio priorities
- Participate in review and validation sessions
- Provide stakeholder attendance for final readout

## Assumptions

- Input data is complete enough for recommendation drafting
- Assessment outputs are draft recommendations, not automatic migration execution plans
- Final customer-ready conclusions are validated with architect review

## Out of Scope (Base Offer)

- Full migration implementation
- Detailed application code remediation plans
- Production cutover execution support
- Security/compliance certification artifacts unless separately scoped

## Success Criteria

- Assessment delivered within agreed timeline
- Recommendation coverage achieved for in-scope workloads
- Architect-approved output quality before customer handoff
- Actionable modernization backlog identified

## Why This Service

This service creates a practical bridge between raw inventory and an executable modernization plan. It accelerates the early assessment phase while preserving expert governance for high-stakes decisions.
